# Future Features Roadmap

Axolotly's product roadmap focuses on enhancing parental control capabilities, improving user experience, and expanding platform reach.

## Phase 1: Core Enhancements (Q1-Q2 2026)

### Screen Time Management
**Description:** Daily time limits and bedtime schedules

**Features:**
- Set daily screen time limits per device
- Scheduled bedtimes (device locks automatically)
- Time extensions (temporary overrides)
- Weekly usage reports

**Benefit:** Complete digital wellness control beyond content curation

### Multi-Parent Access
**Description:** Share family account with co-parents

**Features:**
- Invite additional parents to family account
- Shared policy management
- Activity notifications to all parents
- Permission levels (admin vs. viewer)

**Benefit:** Better co-parenting support, especially for divorced families

### Usage Analytics
**Description:** Detailed reports on content consumption

**Features:**
- Most-watched content per child
- Daily/weekly/monthly screen time charts
- Content discovery patterns
- Export reports for review

**Benefit:** Data-driven insights for parents

## Phase 2: Advanced Features (Q3-Q4 2026)

### Episode-Level Deep Links
**Description:** Direct links to specific episodes, not just S1E1

**Features:**
- Multi-episode deep link database
- Crowdsourced episode URLs
- Episode selection in launcher
- Resume playback from last position

**Benefit:** Complete control over episode access

### Offline Mode
**Description:** Download content for offline viewing

**Features:**
- Download approved titles to device
- Offline content playback
- Automatic sync when online
- Storage management

**Benefit:** Travel-friendly, no internet required

### AI-Powered Content Tagging
**Description:** Machine learning for automated tag accuracy

**Features:**
- Analyze plot summaries with NLP
- Detect nuanced themes automatically
- Improve tag accuracy over time
- Community voting on tag correctness

**Benefit:** More accurate, comprehensive tagging

### Custom Tags
**Description:** Family-specific content descriptors

**Features:**
- Create personal tags (e.g., "triggers-anxiety", "family-favorites")
- Apply to any content
- Share custom tags with community
- Import tag sets from trusted sources

**Benefit:** Ultimate personalization for unique family needs

## Phase 3: Platform Expansion (2027)

### iOS Support
**Description:** Kids launcher for iPhone and iPad

**Features:**
- iOS launcher app
- Deep link support for iOS streaming apps
- Profile-based access control
- Same policy management as Android

**Benefit:** Reach 50% of mobile market

### Web-Based Launcher
**Description:** Browser-based kids interface for computers

**Features:**
- Chromebook compatibility
- Windows/Mac browser launcher
- Same content grid interface
- Keyboard navigation

**Benefit:** Extends control to computers and school devices

### Smart TV Apps
**Description:** Native apps for smart TVs

**Features:**
- Roku, Fire TV, Apple TV apps
- Remote-friendly navigation
- Family mode activation
- Same policy enforcement

**Benefit:** Living room content control

### Gaming Console Support
**Description:** Launcher for PlayStation, Xbox, Nintendo Switch

**Features:**
- Game library curation
- Age-appropriate game filtering
- Playtime limits
- Purchase request system

**Benefit:** Complete digital ecosystem control

## Phase 4: Enterprise & B2B (2027-2028)

### School/Institutional Licensing
**Description:** Bulk deployment for educational institutions

**Features:**
- Multi-school administration
- Teacher account management
- Student profile bulk creation
- Curriculum-aligned content filtering
- FERPA compliance

**Benefit:** Educational market revenue stream

### API for Third-Party Developers
**Description:** Open API for integrations

**Features:**
- RESTful API with authentication
- Webhook notifications
- Tag database access
- White-label deployment options

**Benefit:** Developer ecosystem, expanded use cases

### Telehealth Integration
**Description:** Partner with pediatricians and therapists

**Features:**
- Therapist-recommended content lists
- Clinical tagging for specific conditions (ADHD, anxiety, etc.)
- Progress tracking for therapeutic interventions
- HIPAA-compliant data sharing

**Benefit:** Medical professional endorsement, clinical validation

## Feature Prioritization Criteria

**Decision Matrix:**

1. **User Impact** - How many users benefit?
2. **Development Effort** - How complex to build?
3. **Revenue Potential** - Does it drive subscriptions?
4. **Competitive Advantage** - Unique differentiator?
5. **Market Demand** - How often requested?

**Priority Scoring:**
- P0 (Critical): Must-have for core value prop
- P1 (High): Significant user demand, competitive necessity
- P2 (Medium): Nice-to-have, improves experience
- P3 (Low): Experimental, research phase

## User Feedback Loop

**Continuous Input:**
- In-app feedback forms
- Parent community surveys
- Feature request voting
- Beta testing programs
- Usage data analysis

**Quarterly Roadmap Reviews:**
- Adjust priorities based on feedback
- Add highly-requested features
- Remove low-adoption features
- Iterate on existing capabilities

---

Axolotly's roadmap balances innovation with user needs, ensuring the platform evolves to meet changing family media consumption patterns while maintaining its core mission of empowering parents.
